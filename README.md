# vdashboard

[后端接口参考](https://github.com/vbeats/vcloud)

- `vue3` `typescript` `vuex` `i18n` `ant-design-vue` `webpack`
- `dev`模式下为了加快编译速度未做优化
- `webpack`仅在`build production`时优化资源大小 详见`vue.config.js`配置

### run

```bash
yarn install

yarn serve 

yarn build
```
